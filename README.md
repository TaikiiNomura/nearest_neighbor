# nearest_neighbor
Google Colab上で動作する、画像にモザイク処理を適用するPythonスクリプトです。
画像の元の解像度（ピクセルサイズ）を維持したまま、全体にモザイク効果をかけます。

## 動作環境
- Google Colab (Python 3.x)
- ライブラリ: `Pillow` (PIL)

## 使い方

1. Google Colabのノートブックを開きます。
2. コードセルにスクリプトを貼り付けます。
3. 必要に応じて `apply_mosaic(mosaic_level=15)` の数値を変更してモザイクの粗さを調整します。
    - 推奨値: `10` 〜 `20`（数値が大きいほど粗くなります）
4. セルの再生ボタン（実行ボタン）を押します。
5. 「ファイル選択」ボタンが表示されるので、PC内の画像をアップロードします。
6. 処理が完了すると、`元のファイル名_mosaic.拡張子` という名前で画像が自動ダウンロードされます。

## 処理の仕組み
1. 画像を指定の割合（例: 1/15）に縮小します（`Image.NEAREST`）。
2. 縮小した画像を元の解像度まで引き伸ばします（`Image.NEAREST`）。
3. これにより、ピクセルが拡大され、擬似的なモザイク効果が生まれます。
